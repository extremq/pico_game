# Pico Game
This game is using a [ST7735 128x128](https://www.optimusdigital.ro/en/lcds/3552-modul-lcd-de-144-cu-spi-i-controller-st7735-128x128-px.html) screen
as its display and is controlled by a generic joystick.

Work in progress.
